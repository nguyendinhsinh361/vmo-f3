DATABASE: PROGRESS SQL
+ Thêm một dòng mới: INSERT INTO tableName (key1, key2, ...) VALUES(value1, value2, ...);
+ Tìm kiếm tất cả các dòng có key = value: SELECT * FROM tableName kiHieuTable WHERE key = 'value';
+ Tìm kiếm sắp xếp theo thứ tự
 - Tăng dần: SELECT * FROM tableName kiHieuTable:(a) WHERE a.user_id = 'value' ORDER BY TENCOT ASC
 - Tăng dần: SELECT * FROM tableName kiHieuTable:(a) WHERE a.user_id = 'value' ORDER BY TENCOT DESC
